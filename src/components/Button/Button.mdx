A button can be have multiple colors, sizes and variants:

```typescript jsx
<Box>
  <SimpleGrid columns={5} gap={4} mb={2}>
    <Button size="small">Button</Button>
    <Button size="small" variantColor="teal-500">
      Button
    </Button>
    <Button size="small" variant="outline">
      Button
    </Button>
    <Button size="small" variant="outline" variantColor="teal-500">
      Button
    </Button>
    <Button size="small" variant="outline" variantColor="navyblue-300">
      Button
    </Button>
  </SimpleGrid>
  <SimpleGrid columns={5} gap={4} mb={2}>
    <Button size="medium">Button</Button>
    <Button size="medium" variantColor="teal-500">
      Button
    </Button>
    <Button size="medium" variant="outline">
      Button
    </Button>
    <Button size="medium" variant="outline" variantColor="teal-500">
      Button
    </Button>
    <Button size="medium" variant="outline" variantColor="navyblue-300">
      Button
    </Button>
  </SimpleGrid>
  <SimpleGrid columns={5} gap={4}>
    <Button>Button</Button>
    <Button variantColor="teal-500">Button</Button>
    <Button variant="outline">Button</Button>
    <Button variant="outline" variantColor="teal-500">
      Button
    </Button>
    <Button variant="outline" variantColor="navyblue-300">
      Button
    </Button>
  </SimpleGrid>
</Box>
```

A button can contain an icon:

```typescript jsx
<SimpleGrid columns={2} gap={2}>
  <Button icon="add-user">Button</Button>
  <Button icon="arrow-back" iconAlignment="right">
    Button
  </Button>
</SimpleGrid>
```

A button be marked as loading:

```typescript jsx
<SimpleGrid columns={2} gap={2}>
  <Button loading variantColor="teal-500">
    Button
  </Button>
  <Button loading disabled variantColor="teal-500">
    Button
  </Button>
</SimpleGrid>
```

A button can occupy the full width oof its parent:

```typescript jsx
<Button variantColor="teal-500" fullWidth>
  Button
</Button>
```
