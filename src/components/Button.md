A button can be a pill:

```jsx harmony
<Button pShape="pill" pVariant="primary">
  Hey there
</Button>
```

A button can be a rect:

```jsx harmony
<Button pShape="rect" pVariant="default">
  Hey there
</Button>
```

A button can be small:

```jsx harmony
<Button pShape="pill" pVariant="primary" pSize="small">
  Hey there
</Button>
```
