An input can be valid:

```typescript jsx
const Example = () => {
  const [value, setValue] = React.useState('');
  return (
    <TextInput
      truncated
      label="Name"
      name="hi"
      placeholder="Type something..."
      value={value}
      onChange={e => setValue(e.target.value)}
    />
  );
};

<Example />;
```

An input can be invalid:

```typescript jsx
<TextInput label="Name" name="hi" placeholder="Type something..." value="Invalid Value" invalid />
```

An input can have an icon:

```typescript jsx
const [value, setValue] = React.useState('hey');
<TextInput
  label="Name"
  name="hi"
  value={value}
  placeholder="Type something..."
  iconProps={{ sx: { color: 'navyblue-300', opacity: 0.8 } }}
  onChange={e => setValue(e.target.value)}
  icon="dashboard-alt"
/>;
```

The input icon can get aligned

```typescript jsx
const [value, setValue] = React.useState('hey');
<TextInput
  label="Name"
  name="hi"
  iconAlignment="left"
  value={value}
  placeholder="Type something..."
  iconProps={{ sx: { color: 'yellow-400', opacity: 0.8 } }}
  onChange={e => setValue(e.target.value)}
  icon="log-source"
/>;
```

An input can be disabled:

```typescript jsx
<TextInput label="Name" name="hi" disabled placeholder="Type something..." value="Disabled Value" />
```
