Alerts can be simple:

```typescript jsx
<Box width={1}>
  <Box mb={3}>
    <Alert type="success" mb={3} title="This is a success alert" />
  </Box>
  <Box mb={3}>
    <Alert type="info" mb={3} title="This is an info alert" />
  </Box>
  <Box mb={3}>
    <Alert type="warning" mb={3} title="This is a warning alert" />
  </Box>
  <Box mb={3}>
    <Alert type="error" title="This is an error alert" />
  </Box>
  <Box>
    <Alert type="default" title="This is the default alert" />
  </Box>
</Box>
```

Alerts can be opaque:

```typescript jsx
<Box width={1}>
  <Box mb={3}>
    <Alert type="success" variant="opaque" mb={3} title="This is a success alert" />
  </Box>
  <Box mb={3}>
    <Alert type="info" variant="opaque" mb={3} title="This is an info alert" />
  </Box>
  <Box mb={3}>
    <Alert type="warning" variant="opaque" mb={3} title="This is a warning alert" />
  </Box>
  <Box mb={3}>
    <Alert type="error" variant="opaque" title="This is an error alert" />
  </Box>
  <Box>
    <Alert type="default" variant="opaque" title="This is the default alert" />
  </Box>
</Box>
```

Alerts can have actions associated with them:

```typescript jsx
<Box>
  <Box mb={3}>
    <Alert
      type="warning"
      title="This is a warning alert"
      description="This is a nice description of the title"
      actions={<Button onClick={() => alert('Boom!')}>I Alert</Button>}
    />
  </Box>
  <Box>
    <Alert
      type="error"
      title="This is an error alert"
      description="This is a nice description of the title"
      actions={({ close }) => <Button onClick={close}>I Close</Button>}
    />
  </Box>
</Box>
```
