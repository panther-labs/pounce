Alerts can be simple:

```typescript jsx
<Box width={1}>
  <Box mb={3}>
    <Alert alertType="success" mb={3} title="This is a success alert" />
  </Box>
  <Box mb={3}>
    <Alert alertType="info" mb={3} title="This is an info alert" />
  </Box>
  <Box mb={3}>
    <Alert alertType="warning" mb={3} title="This is a warning alert" />
  </Box>
  <Box mb={3}>
    <Alert alertType="error" title="This is an error alert" />
  </Box>
  <Box>
    <Alert alertType="default" title="This is the default alert" />
  </Box>
</Box>
```

Alerts can be opaque:

```typescript jsx
<Box width={1}>

    <Alert alertType="success" variant="opaque" mb={3} title="This is a success alert" />
  </Box>
  <Box mb={3}>
    <Alert alertType="info" variant="opaque" mb={3} title="This is an info alert" />
  </Box>
  <Box mb={3}>
    <Alert alertType="warning" variant="opaque" mb={3} title="This is a warning alert" />
  </Box>
  <Box mb={3}>
    <Alert alertType="error" variant="opaque" title="This is an error alert" />
  </Box>
  <Box>
    <Alert alertType="default" variant="opaque" title="This is the default alert" />
  </Box>
</Box>
```

Alerts can have actions associated with them:

```typescript jsx
<Box>
  <Box mb={3}>
    <Alert
      alertType="warning"
      title="This is a warning alert"
      description="This is a nice description of the title"
      discardable
      actions={
        <Button alertTypeColor="red" onClick={() => alert('Boom!')}>
          I Alert
        </Button>
      }
    />
  </Box>
  <Box>
    <Alert
      alertType="error"
      title="This is an error alert"
      description="This is a nice description of the title"
      discardable
      actions={({ close }) => <Button onClick={close}>I Close</Button>}
    />
  </Box>
</Box>
```
