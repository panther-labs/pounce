In order to use snackbars, you need to get access to the `pushSnackbar` function. To do that, you
need to use React's Context API. You can either render a `SnackbarConsumer` component or simply
import & include a `useSnackbar` call in your functional component.

The context currently exposes an object containing the `pushSnackbar` function which accepts a single
object argument containing the same exact props that you would normally be passing to the

<a href="/#/Alert">Alert</a> component.

```shell
{ pushSnackbar: (data: AlertProps) => void }
```

A Snackbar is just a re-export of the Alert in a more digestable fashion with the `discardable`
option enable by default (overridable).

You can push as many snackbars as you want:

```typescript jsx
const Example = () => {
  const { pushSnackbar } = useSnackbar();

  return (
    <Button
      size="large"
      variant="primary"
      onClick={() =>
        pushSnackbar({ variant: 'success', title: 'Everything went perfectly', icon: 'check' })
      }
    >
      Click me a lot of times
    </Button>
  );
};

<SnackbarProvider>
  <Example />
</SnackbarProvider>;
```

You can tweak the duration of the snackbar through the `duration` prop:

```typescript jsx
const Example = () => {
  const { pushSnackbar } = useSnackbar();

  return (
    <Button
      variant="default"
      onClick={() =>
        pushSnackbar({
          variant: 'error',
          title: 'Everything went wrong',
          icon: 'remove',
          duration: 1000,
        })
      }
    >
      I disappear faster
    </Button>
  );
};

<SnackbarProvider>
  <Example />
</SnackbarProvider>;
```

You can change the position (default is bottom left):

```typescript jsx
const Example = () => {
  const { pushSnackbar } = useSnackbar();

  return (
    <Flex align="center" justify="space-between" direction="column" style={{ width: '100%' }}>
      <Flex align="center" justify="space-between" direction="row" mb={6} style={{ width: '100%' }}>
        <Button
          variant="primary"
          onClick={() =>
            pushSnackbar({
              variant: 'success',
              title: 'It works!',
              position: 'top-left',
            })
          }
        >
          Top Left
        </Button>
        <Button
          variant="primary"
          onClick={() =>
            pushSnackbar({
              variant: 'success',
              title: 'It works!',
              position: 'top-middle',
            })
          }
        >
          Top Middle
        </Button>
        <Button
          variant="primary"
          onClick={() =>
            pushSnackbar({
              variant: 'success',
              title: 'It works!',
              position: 'top-right',
            })
          }
        >
          Top Right
        </Button>
      </Flex>
      <Flex align="center" justify="space-between" direction="row" style={{ width: '100%' }}>
        <Button
          variant="primary"
          onClick={() =>
            pushSnackbar({
              variant: 'success',
              title: 'It works!',
              position: 'bottom-left',
            })
          }
        >
          Bottom Left
        </Button>
        <Button
          variant="primary"
          onClick={() =>
            pushSnackbar({
              variant: 'success',
              title: 'It works!',
              position: 'bottom-middle',
            })
          }
        >
          Bottom Middle
        </Button>
        <Button
          variant="primary"
          onClick={() =>
            pushSnackbar({
              variant: 'success',
              title: 'It works!',
              position: 'bottom-right',
            })
          }
        >
          Bottom Right
        </Button>
      </Flex>
    </Flex>
  );
};

<SnackbarProvider>
  <Example />
</SnackbarProvider>;
```
