By default the Breadcrumbs will render anchor tags (`a`):

```typescript jsx
const items = [
  {
    title: 'First page',
    href: '/#/Box',
  },
  {
    title: 'Second page',
    href: '/#/Flex',
  },
  {
    title: <MyCustomTitle />,
    href: '/#/MyCustomTitle',
  },
];

<Breadcrumbs items={items} />;
```
